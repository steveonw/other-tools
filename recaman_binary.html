Exploring how binary patterns modify the classic sequence

Experiment: Select a mode below to see how adding binary signals changes the Recamán sequence behavior!
🌀 Original Recamán
a(n) = a(n-1) - n if positive and new, else a(n-1) + n
➕ Add Binary Value
Result + binary[n] (adds 0 or 1 to each term)
🔄 Force Direction
Binary determines add/subtract (overrides normal rule)
✖️ Multiply Step
n × (1 + binary[n]) — step size varies by signal
⚡ XOR Perturbation
Result XOR binary[n] (bitwise modification)
🎯 Conditional Apply
Only apply Recamán rule when binary[n] = 1
